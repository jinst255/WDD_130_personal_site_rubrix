## About Rubrix

Rubrix is a Chrome extension that integrates with Canvas LMS to give students AI-powered, rubric-aligned feedback before they submit. Built with the Anthropic API, Convex, TypeScript, and Stripe.

> Rubrix itself is developed by Weiss Solutions LLC (Mikael, Justin, & Talmage as developers) and is not affiliated with Instructure or Canvas LMS.
